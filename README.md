# Bank Marketing Campaign Analysis

## 📊 Project Overview
This project analyzes a bank marketing dataset to understand customer behavior and identify factors affecting term deposit subscriptions. The dashboard was created using Power BI.

## 🎯 Objective
To analyze customer data and identify which factors such as age, job, education, and financial status influence subscription decisions.

## 📈 Key Insights
- Certain job categories have higher subscription rates.
- Customers with higher account balances are more likely to subscribe.
- Subscription behavior varies across age groups and education levels.
- Most customers did not subscribe, indicating the need for better targeting.

## 🛠 Tools Used
- Power BI
- Excel

## 📁 Files
- Bank_Marketing_Analysis.pdf (Dashboard Report)

## 📚 Data Source
UCI Machine Learning Repository  
https://archive.ics.uci.edu/ml/datasets/bank+marketing

## 📌 Conclusion
The analysis shows that customer demographics and financial factors influence subscription decisions. These insights can help banks improve marketing strategies and increase subscription rates.
